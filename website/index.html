Features

Built for the Next Generation

Every feature is designed for one purpose — to make you faster and more creative.

⚡

Blazing Rust Performance

Built from the ground up in Rust. Instant startup, minimal memory, no lag — even in massive monorepos.

🧠

Deep AI Integration

Not just autocomplete. Chat, edit, plan, review, debug, and research modes that understand your entire project.

🎨

Beautiful Custom Themes

From dynamic gradients to neon glow. Every pixel is yours to customize with a powerful theming engine.

🔌

Multi-Provider AI

OpenAI, Anthropic, OpenRouter, Ollama, DeepSeek, Qwen, Grok & more. All configurable from one settings panel.

🛠️

Agent Tool System

Your AI can read files, search code, edit text, run terminal commands, and browse the web — all autonomously.

🌐

Cross-Platform

Native on Linux, macOS, and Windows. One codebase, flawless experience everywhere.

AI Modes

Six Minds, One Editor

Each mode is a specialized AI personality tuned for a specific task.

💬

Chat

General-purpose AI conversation. Ask questions, get explanations, explore ideas.

✏️

Edit

Describe the change and the agent reads your project, plans the edit, and applies it.

📋

Plan

Analyzes your codebase and generates a step-by-step plan before writing any code.

🔍

Review

Reads your code, identifies issues, and suggests improvements with explanations.

🐛

Debug

Reads error context, runs diagnostics, and searches for known solutions — automatically.

📚

Research

Searches your workspace and external sources to answer deep technical questions.
